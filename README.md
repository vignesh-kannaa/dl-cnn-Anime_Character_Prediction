# dl-cnn-Anime_Character_Prediction

Image classification with Deep Learning Convolutional Neural Network


Upload any one of the following characters from One Piece and it predicts the correct output with accuracy of 80%: ['Brook','Chopper','Franky','Luffy''Nami''Robin','Sanji','Usopp','Zoro']
